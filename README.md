# Dead by Daylight (DBD) moonwalk python script
A simple and efficient tool to automate the "moonwalk" technique in the game Dead by Daylight.

## Features
Automation of W, A, S, D, and SHIFT keys  
Smooth alternation between D and A keys

## How this script works
It will rotate your character 180 degrees by pressing (W, A , S and D) keys in a sequence.  
Then pressing the A and D keys in a smooth timed sequence while holding shift and W keys to initiate the moonwalk.  
By pressing the left side button on the mouse you will toggle the moonwalk function on and off.

## Installation
1. Make sure you have Python 3.6+ installed
2. Install dependencies:
```
pip install pynput pillow
```
## Run the script:
```
python dbd_moonwalk.py
```
